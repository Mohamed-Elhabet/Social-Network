# Social Network 

### Simple network application using : 

* Python Programming Language 
* Django Framework 
* Sqlite
* Javascript, Ajax 


### Simple network application that allow:
  * register new user 
  * login / logout user 
  * add posts (content, image)
  * add comments for post
  * like / unlike post 
  * add / remove friends 
  * update / delete own posts
